# SongSix
Texts written after Retrospective 2024

My contribution to understanding the notation of time is substantially complete, and has been summarized in **Retrospective 2024** and the preceding writings.<br>

The texts that follow are therefore "SongSix": They correspond to *Song Six*, which follows *Act Two*, in which Clytæmnestra comes back as a ghost to haunt *"the denizens of hell"*. (*"What, slumber ye?..."*)<br>
There are actually no restrictions on what she can say...<br>

## repository
This is a private repository in the _**writings/SongSix**_ folder in https://james-ingram-act-two.de used while developing new texts.<br>
Each text/essay is developed in its own folder (using GitHub branching etc.) and, when complete, published in the corresponding folder on my website.

### branches

**main**: This contains the final versions of all texts as published on my website.<br>
New texts should be created in their own folders and branches, and merged into the **main** branch when published.
